# ASI Changelog

## v0.1 — February 2026

Initial draft release of Agent Skill Identity (ASI).

Includes:

- Ed25519-based identity primitive
- DID:key identity format
- Publisher bundle signing
- Invocation envelope signing
- JCS (RFC 8785) canonicalization requirement
- Domain-separated signing inputs
- Symlink and undeclared file rejection
- Node reference SDK
- Test vectors and example bundles
